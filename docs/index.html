Te ofrecemos cursos que te ayudarán a mejorar tus habilidades, las cuales te permitirán desarrollar un sitio web mediante el uso de la tecnología Front-end.

Para el desarrollo front-end se utilizan diferentes herramientas como HTML, CSS, entre otros.
Si nunca has escuhado sobre éstas, no te preocupes que aquí te damos una breve explicación, si quieres saber más puedes ir a la sección ilústrate, o bien, inscribete a uno de los cursos que ofrecemos dando clic aquí.

HTML

Lenguaje de Maracado de Hipertextos es usado para crear documentos en la web.
Se utiliza para organizar el contenido, por lo que se toma como la estructura de una página web.

CSS

Hojas de estilo en cascada se usa para la presentación del contenido de una página web.
Se encarga de dar diseño, estilo o aspecto visual a cualquier elemento de un sitio web.
